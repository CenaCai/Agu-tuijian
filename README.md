# Agu-tuijian（A 股推荐）

每日北京时间 20:00 自动运行选股策略，结果以 JSON 格式保存在 `results/` 目录。

## 策略设计：5日5%模型

专为**挂单操作模式**设计，核心思路是只筛选高确定性的短期机会：

| 参数 | 说明 |
|------|------|
| 买入价 | 收盘价 - 0.5%（低吸挂单） |
| 卖出价 | 买入价 + 5%（目标止盈） |
| 止损价 | 买入价 - 2%（止损线） |
| 持有期 | 5 个交易日 |
| 盈亏比 | 2.5（赚5%，止损亏2%） |

## 筛选流程

1. **基础筛选** — 市值>50亿，成交额>5000万，价格3-100元
2. **排除规则** — 连续跌>3天、有跌停、振幅>8%、当日涨幅>5%
3. **动量确认** — 5日涨幅>0%，20日涨幅>3%，量能不萎缩
4. **波动率适配** — 5日ATR在1%-3%之间（能到5%但不暴跌）
5. **趋势支撑** — 站上MA20且偏离<3%，MA30斜率向上
6. **综合评分** — 多因子加权排序，取前10只

## 输出字段

| 字段 | 说明 |
|------|------|
| `code` | 股票代码 |
| `name` | 股票中文名称 |
| `buy_price` | 建议买入挂单价 |
| `sell_price` | 目标卖出挂单价（+5%） |
| `stop_price` | 止损挂单价（-2%） |
| `hold_days` | 持有天数（固定5天） |
| `score` | 综合评分（满分~103） |
| `metrics` | 详细指标（5日涨幅、ATR、量比等） |
| `detail` | 策略详情摘要 |

## 文件说明

- `.github/workflows/daily_stock_pick.yml` — GitHub Actions 工作流
- `scripts/run_selection.py` — 选股脚本
- `requirements.txt` — Python 依赖

## 触发方式

- 自动：每天北京时间 20:00
- 手动：在 GitHub 仓库的 **Actions** 选项卡点击 **Run workflow**

## 重要说明

本策略条件较为严格，**某些交易日可能选不出股票**。这是正常且预期的行为 — 市场没有高确定性机会时，空仓等待是最优选择。不要为了"有推荐"而降低标准。

## 配置调整

在 `scripts/run_selection.py` 顶部可以修改筛选参数，如 `TARGET_RETURN`、`STOP_LOSS_PCT`、`MIN_MARKET_CAP` 等。
